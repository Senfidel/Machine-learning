## Aplication de prediction de salaires des data scientists
Définition des variables
work_year : L’année où le salaire a été versé. experience_leve : Le niveau d’expérience dans l’emploi au cours de l’année avec les valeurs possibles suivantes : EN Débutant / Junior MI Niveau intermédiaire / Intermédiaire SE Niveau supérieur / Expert EX Niveau exécutif / Directeur employment_type : Le type d’emploi pour le poste : PT Temps partiel FT Temps plein CT Contrat FL Freelance job_title : Le rôle occupé au cours de l’année. salary : Le montant total du salaire brut versé. salary_currency : La devise du salaire versé sous la forme d’un code de devise ISO 4217. salary_in_usd : Le salaire en USD (taux de change divisé par le taux de change moyen de l’année concernée via fxdata.foorilla.com). employee_residence : Pays de résidence principale de l’employé au cours de l’année de travail sous la forme d’un code de pays ISO 3166. remote_ratio : :Le montant total du travail effectué à distance, les valeurs possibles sont les suivantes : 0 - Pas de travail à distance (moins de 20%), 50 - Partiellement à distance, 100 - Entièrement à distance (plus de 80%) company_location : Le pays du siège social ou de la branche contractante de l’employeur sous la forme d’un code pays ISO 3166. company_size : Le nombre moyen de personnes ayant travaillé pour l’entreprise au cours de l’année : S moins de 50 salariés (petit) M 50 à 250 salariés (moyen) L plus de 250 salariés (grand)
